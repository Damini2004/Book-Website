# **App Name**: MPH Online

## Core Features:

- Journal Listings: Display journal information, including titles, descriptions, and links to journal websites.
- Book Catalog: Showcase books with categories, new releases, and bestsellers.
- Book Proposal Submission: Enable authors to submit book proposals via a detailed form, including author information, book details, and sample chapters.
- Content Generation Tool: Generate a book summary of uploaded files using AI. This tool provides prospective authors a preview of their ideas when writing their proposals.
- Newsletter Subscription: Allow users to subscribe to a monthly newsletter for new issues and book releases.

## Style Guidelines:

- Primary color: Light greenish blue (#A7D1AB), suggestive of sustainable agriculture and open access.
- Background color: Very pale greenish blue (#F0F4F1). Background should offer contrast without being distracting.
- Accent color: muted gold (#C2B280). This communicates knowledge, authority and publishing.
- Headline font: 'Playfair', serif, for titles and headings to convey elegance and authority; body font: 'PT Sans', sans-serif, for body text to ensure readability.
- Use clear, professional icons related to agriculture, entomology, and publishing.
- Maintain a clean and structured layout with clear navigation. Use a grid system for consistent spacing and alignment.
- Use subtle animations and transitions to enhance user experience without being distracting.
